# homebrew-kaleidot725

Homebrew tap for kaleidot725's applications.

## Installation

### AdbPad

GUI tool for Android app testing using ADB.

```bash
brew tap kaleidot725/app
brew install --cask adbpad
```

### Update

```bash
brew update
brew upgrade --cask adbpad
```

### Uninstall

```bash
brew uninstall --cask adbpad
```
